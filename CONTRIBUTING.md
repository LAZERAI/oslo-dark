# Contributing

## Reporting issues

[Open an issue](https://github.com/LAZERAI/oslo-dark/issues) with:

- The language/file type where something looks wrong
- A screenshot showing the problem
- Your VS Code version

## Making changes

1. Fork and clone the repo
2. Press F5 in VS Code to open the Extension Development Host
3. Edit `themes/oslo-dark-color-theme.json`
4. Check your changes in JS, Python, JSON, and Markdown files at minimum
5. Open a PR with a short explanation

Keep new colors consistent with the existing palette. The theme is blue-dominant with lavender, green, and warm sand accents.
